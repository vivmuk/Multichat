Everything you need to build with Venice

Venice provides a privacy-first AI API with no data retention. It's OpenAI-compatible — just swap the base URL. Supports text, image, audio, music, video, embedding, and more.

Chat Completions
POST /chat/completions — OpenAI-compatible
ParameterTypeRequiredDescription
modelstringRequiredModel ID, e.g. zai-org-glm-4.7, llama-3.3-70b
messagesarrayRequiredArray of message objects with role (system/user/assistant/tool) and content
temperaturenumberOptional0–2. Controls randomness. Default: 1
max_completion_tokensintegerOptionalMaximum output tokens to generate
streambooleanOptionalStream tokens as SSE. Default: false
top_pnumberOptionalNucleus sampling (0–1)
top_kintegerOptionalLimit sampling to top-k tokens
frequency_penaltynumberOptional-2 to 2. Reduces token repetition
presence_penaltynumberOptional-2 to 2. Encourages new topics
toolsarrayOptionalFunction definitions for tool calling
reasoning_effortstringOptionalnone | minimal | low | medium | high | xhigh | max
venice_parametersobjectOptionalVenice-specific extensions (see below)
terminalcURL — Basic Chat 
curl --request POST \
  --url https://api.venice.ai/api/v1/chat/completions \
  --header 'Authorization: Bearer YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "model": "zai-org-glm-4.7",
    "messages": [
      {"role": "system", "content": "You are a helpful assistant."},
      {"role": "user", "content": "Explain quantum computing in simple terms."}
    ],
    "temperature": 0.7,
    "max_completion_tokens": 500
  }'
codePython — Basic Chat 
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.venice.ai/api/v1"
)

response = client.chat.completions.create(
    model="zai-org-glm-4.7",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Explain quantum computing in simple terms."}
    ],
    temperature=0.7,
    max_tokens=500
)

print(response.choices[0].message.content)

codePython — Streaming 
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_API_KEY",
    base_url="https://api.venice.ai/api/v1"
)

stream = client.chat.completions.create(
    model="zai-org-glm-4.7",
    messages=[{"role": "user", "content": "Write a short poem about the ocean."}],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="", flush=True)

Venice-Specific Parameters

Pass a venice_parameters object alongside standard parameters for Venice-specific features:

ParameterTypeDescription
enable_web_searchstringauto | on | off — Adds live web search results (+$0.01/request)
enable_web_citationsbooleanInclude source citations in response
enable_web_scrapingbooleanScrape full page content for URLs in prompt
enable_e2eebooleanEnd-to-end encryption for the request
character_slugstringUse a Venice AI character persona (e.g. venice)
disable_thinkingbooleanStrip internal reasoning/thinking tokens from response

heightintegerOptional1–1280. Default: 1024

return_binarybooleanOptionalReturn raw image bytes instead of base64 JSON
safe_modebooleanOptionalBlur adult content. Default: true
hide_watermarkbooleanOptionalAttempt to hide Venice watermark
embed_exif_metadatabooleanOptionalEmbed generation params in image EXIF data
style_presetstringOptionalApply a style preset to the generation
aspect_ratiostringOptionale.g. 1:1, 16:9, 9:16

codePython — Generate & Save Image 
import requests, base64

response = requests.post(
    "https://api.venice.ai/api/v1/image/generate",
    headers={"Authorization": "Bearer YOUR_API_KEY"},
    json={
        "model": "flux-2-pro",
        "prompt": "A futuristic city at night, neon lights, cyberpunk",
        "width": 1024,
        "height": 1024,
        "steps": 20,
        "seed": 42
    }
)

data = response.json()
image_data = base64.b64decode(data["images"][0])

with open("output.webp", "wb") as f:
    f.write(image_data)

print(f"Image saved. Generation time: {data['timing']['total']}ms")

SDK Quickstart
Use existing OpenAI SDKs — just change the base URL
lightbulb
Drop-in replacement: Venice is OpenAI API-compatible. If you already use the OpenAI Python or JS SDK, just set base_url="https://api.venice.ai/api/v1" and your Venice API key. No other changes needed.

codePython — Full Setup 
import os
from openai import OpenAI

client = OpenAI(
    api_key=os.environ.get("VENICE_API_KEY"),
    base_url="https://api.venice.ai/api/v1"
)

# Chat
chat = client.chat.completions.create(
    model="zai-org-glm-4.7",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(chat.choices[0].message.content)

# List models
models = client.models.list()
for m in models.data:
    print(m.id)
javascriptTypeScript — Full Setup 
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: process.env.VENICE_API_KEY,
  baseURL: 'https://api.venice.ai/api/v1'
});

async function main() {
  // Chat
  const chat = await client.chat.completions.create({
    model: 'zai-org-glm-4.7',
    messages: [{ role: 'user', content: 'Hello!' }]
  });
  console.log(chat.choices[0].message.content);

  // List models
  const models = await client.models.list();
  models.data.forEach(m => console.log(m.id));
}

main();

Error Reference
HTTP status codes returned by the Venice API
StatusMeaningHow to handle
200SuccessParse response normally
400Bad Request — invalid parametersCheck required fields and value ranges
401Unauthorized — invalid/missing API key or Pro access requiredVerify API key; upgrade if model requires Pro
402Payment Required — insufficient x402 USDC balanceTop up your x402 wallet balance
429Rate Limit ExceededBack off and retry with exponential delay
500Internal Server Error — inference failedRetry the request
503Service Unavailable — model at capacityRetry after a short delay or use an alternate model
504Gateway Timeout — request took too longUse stream: true for long outputs
